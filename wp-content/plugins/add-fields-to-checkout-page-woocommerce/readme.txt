Add Custom Fields to Checkout & My Account Page WooCommerce

== Description ==

This Plugin will help you achieve following tasks, Developed by <a href="https://themelocation.com">themelocation</a> 

1- Add Registration fields WooCommerce
2- WooCommerce Checkout page customization
3- Add fields to Register form WooCommerce
4- Add custom fields WooCommerce checkout
5- Add product custom field in WooCommerce 
6- User registration form Woocommerce
7- Add fields to My Account WooCommerce page

You can add, Edit, delete, re-arrange custom fields in Billing page, Register page, shipping page and My Account WooCommerce. It is very simple to use for the beginners.

If You need any Help Simple <a href="http://themelocation.com/contact/">Contact us</a>

== Installation ==

1. From the Dashboard:
2. Plugins
3. Add New
4. Search "Checkout Editor WooCommerce"
5. Install Now
6. Activate
7. Go to Dashboard:-> Checkout Editor WooCommerce
