== Description ==
VC Countdown oCoder Free is a free version of VC Countdown oCoder.<br>
Please see the  <a href="http://demo.ocodereducation.com/count-down/" target="_blank">demo here</a>.This add on will help you to create beautiful countdown with visual composer. We prepared many style, you only need to click and change color you want.
 Features

    Create Countdown easily
	Edit Countdown Style 
	Many Countdown on one page

How to use

    Install
	Create Countdown from Visual Composer editor
	Change date
	Change color or background
	Done!!!


== Installation ==

1. Please install and active visual composer plugin
2. Install and active this plugin. If you do not know how to install and active a wordpress plugin, please take a look here:https://www.siteground.com/tutorials/wordpress/wordpress_plugins.htm
3. Please config the plugin here: http://yourhost.com/wp-admin/admin.php?page=vc_contact_us_ocoder%2Fadmin-page.php
Please don't forget to click on "Save Changes" button to save your edit.
4. Go to your page and edit, it's better to use visual composer Front End edit because you can see what change directly.
If you have any question, please feel free to contact us via email trungstormsix@gmail.com
